# backend-api

Единый монолитный Laravel API для информационной системы управления автотранспортом ООО «АЗЫК».

## Основные зоны ответственности

- авторизация через Laravel Sanctum;
- разграничение доступа по ролям;
- план-наряды;
- путевые листы и state machine;
- медосмотры и техосмотры;
- заправки;
- GPS/ГЛОНАСС-имитация;
- OCR-контроль фото одометра через вспомогательный PaddleOCR-контейнер;
- PDF путевых листов;
- отчеты и экспорт Excel;
- WebSocket-события;
- журнал действий.

## Запуск через Docker Compose

```bash
docker compose up --build
```

Локальный PHP/Composer не требуется. Контейнер `backend-setup` сам устанавливает зависимости, создает `.env`, генерирует ключ Laravel, выполняет миграции и seed.

## Важные решения

- Backend не делится на микросервисы.
- Вся бизнес-логика смены проходит через `WaybillStateService`.
- Распознавание одометра вызывается из Laravel через `OdometerControlService`; OCR только предлагает значение, финальное значение сохраняется после подтверждения водителя.
- Мобильное приложение восстанавливает этап workflow через endpoint `/api/mobile/workflow`.
- PDF формируются через Blade-шаблоны в `resources/views/pdf`.
