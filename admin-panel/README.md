# admin-panel

Web-административная панель сотрудников ООО «АЗЫК».

Технологии:

- Vue 3;
- Pinia;
- Vue Router;
- Axios;
- Leaflet + OpenStreetMap;
- Laravel Echo для WebSocket-событий.

## Разделы

- Dashboard;
- Пользователи;
- Водители;
- Автомобили;
- План-наряды;
- Путевые листы;
- Медосмотры;
- Техосмотры;
- Карта;
- История движения;
- Заправки;
- Отчеты;
- Журнал действий;
- Настройки/справочники.

## Запуск через Docker Compose

```bash
docker compose up --build
```

Админка будет доступна на `http://localhost:5173`.

Доступ администратора:

- логин: `admin`;
- пароль: `admin123`.
